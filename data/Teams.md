# Teams

File type: csv

May contain multiple/all assignments or just one

Only one class

- R1: Header
    - C1: First Name
    - C2: Last Name
    - C3: Email address
    - C4: assignment name (points earned)
    - C5: Points (total)
    - C6: Feedback
    - If multiple assignments:
        - C(k): assignment name (points)
        - C(k+1): Points (total)
        - C(k+2): Feedback
